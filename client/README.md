# Video Call App

## Tech Stack
- Next.js (Frontend)
- Node.js + Express (Backend)
- Twilio Video API

## Features
- Join room
- Leave room
- Camera preview
- Connection status

## Setup

### Frontend
cd client
npm install
npm run dev

### Backend
cd server
npm install
node index.js

## Environment Variables

TWILIO_ACCOUNT_SID=
TWILIO_API_KEY=
TWILIO_API_SECRET=
